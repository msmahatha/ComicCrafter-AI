📜 1. Introduction
ComicCrafter AI is an AI-powered comic generator that runs locally on MacBook Air (Apple Silicon or Intel) and other compatible devices. It uses: ✅ Llama for AI-generated storytelling ✅ Stable Diffusion (Diffusers) for AI-driven image generation ✅ Streamlit UI for easy comic customization
This manual will guide you through the installation, setup, and usage of ComicCrafter AI.

💻 2. System Requirements
✅ Hardware Requirements
Processor: Intel/AMD CPU or Apple M1/M2 chip
RAM: Minimum 8GB (Recommended: 16GB+)
Storage: At least 10GB free space
GPU: A GPU with at least 4GB VRAM is recommended for faster processing (Optional)
✅ Software Requirements
Operating System: macOS (Monterey/Ventura/Sonoma)
Python Version: 3.10+
Dependencies: diffusers, transformers, torch, sentencepiece, llama-cpp-python, pillow, streamlit

⚙️ 3. Installation Guide
Follow these steps to install ComicCrafter AI on your system.
🔹 Step 1: Clone the Repository
git clone https://github.com/YourUsername/ComicCrafterAI.git
cd ComicCrafterAI
🔹 Step 2: Create a Virtual Environment
python3 -m venv venv
source venv/bin/activate  # (For Mac/Linux)
🔹 Step 3: Install Dependencies
pip install --upgrade pip  
pip install -r requirements.txt
🔹 Step 4: Download Pre-Trained Models
Download the required Llama & Stable Diffusion models and place them in the /models directory.
Llama (Text Generation): Download from Meta AI
Stable Diffusion (Image Generation): Download from Hugging Face

🚀 4. Running the Application
After installation, you can launch ComicCrafter AI with the following command:
streamlit run src/app.py

This will start a local web application in your browser.

🎨 5. Using the ComicCrafter AI Interface
Once the application is running, follow these steps:
🔹 Step 1: Enter Story Prompt
Type a brief story idea in the input box.
Click "Generate Story" to create a script using Llama.
🔹 Step 2: Generate Comic Panels
Click "Generate Comic", and the AI will create illustrations using Stable Diffusion.
Select the manga template to arrange the comic panels.
🔹 Step 3: Customize Your Comic
You can edit the generated text or rearrange images before finalizing.
Choose text bubble styles and panel layout.
🔹 Step 4: Export the Comic
Click "Download Comic" to save it as a PDF or PNG.

🖌️ 6. Adding Custom Manga Templates
To use your own Manga Templates, follow these steps:
🔹 Step 1: Prepare Your Template
Your template should be in PNG/JPG format.
It should have transparent areas for placing generated images.
🔹 Step 2: Add to the assets/templates Directory
Place your template in:
assets/templates/
🔹 Step 3: Select the Template in the UI
In the application, choose "Upload Custom Template".
Select your uploaded manga template and apply it to your comic.

🛠️ 7. Troubleshooting
❌ Issue: Application Fails to Start
Solution: Ensure all dependencies are installed correctly:

pip install -r requirements.txt

❌ Issue: Comic Generation is Too Slow
Solution:
Use a smaller Stable Diffusion model (e.g., SD 1.4 instead of SDXL).
Run on a machine with a GPU.
❌ Issue: Text is Not Matching the Images
Solution:
Use detailed prompts when generating the story.
Manually adjust the generated text before image generation.

❓ 8. FAQ
Q1: Can I run this on Windows?
Yes, but some dependencies may require WSL (Windows Subsystem for Linux).
Q2: Do I need an internet connection?
No, ComicCrafter AI runs entirely offline after setup.
Q3: Can I edit the generated comics?
Yes, you can manually edit text, images, and panel layouts.
